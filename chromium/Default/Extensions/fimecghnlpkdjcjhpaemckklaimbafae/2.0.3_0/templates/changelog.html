Surge has been updated!

Version 2.0 is Here!

Improvements

  • New Feature: Select City By ZIP Code (US Only)
    • Set your weather location by entering the ZIP code of your city
  • New Feature: Show/Hide Temperature Decimal Places
    • Toggle between between a more accurate or more simplistic temperature view
  • Implemented cache layer to improve background update speed and reliability
  • Refactored codebase for improved performance, efficiency, and scalability

Release Notes

  • Surge 2.0 was developed with future releases in mind. Several aspects of the codebase were refactored or rewritten entirely to allow for a faster and more efficient development workflow moving forward. Denoted by the major version increment, an unavoidable consequence of this rewrite was the introduction of breaking changes that affect backward compatibility.
  • Breaking Change: Background Image
    • Your background image may have disappeared. Please reset your background image from the settings page to resolve this issue.

Changelog – Version 1.4.0

Improvements

  • New Feature: Set Background Image Update Interval
    • Choose how often your background image will automatically update from the settings page
    • Default update interval set to every 15 minutes
  • Added additional save/home buttons to the settings page for a better user experience
  • Tweaked card styling for better visual balance

Bug Fixes

  • Fixed edge case where background image updating interval fell out of sync

Changelog – Version 1.3.0

Improvements

  • New Feature: Main Card Styling and Themes
    • Change the colour and transparency of the main card's individual components from the settings page
    • Switch between the built-in themes or create a custom theme of your own
  • Added card styling and wallpaper preview in settings for a better user experience
  • Reconfigured card size for better consistency when seconds are enabled

Bug Fixes

  • Fixed bug where pressing Control + F on a Mac would incorrectly trigger "Ahoy!"

Changelog – Version 1.2.2

Improvements

  • New Feature: "Ahoy!" – A Better Search Bar
    • Activate "Ahoy!" by clicking on search in the top right or by pressing Ctrl/⌘ + F
    • By default, "Ahoy!" will perform a standard Google search with your query
    • Type a keyword before your query to search that site instead (e.g. "youtube trailers", "wolfram graph cosx")
  • Cleaned up code for better performance

Bug Fixes

  • Fixed issue where cursor would show move icon when hovering on top of the weather tooltip
  • Fixed bug where IMDB query would duplicate input on search
  • Fixed ⌘ + F shortcut to launch "Ahoy!" for Mac users

Changelog – Version 1.1.1

Improvements

  • New Feature: Draggable Clock
    • Click and drag anywhere on the clock to reposition it
    • Reset the position of the clock to its default position from the settings page
  • Surge now opens to the settings page instead of the main page on initial install
  • Highlighted "Select search method below" text when search methods are unset to better catch user's attention
  • Made opacity change when options become disabled as a result of enabling other options to improve clarity
  • Added visual cue when pressing save button in settings to better indicate that the action took place

Bug Fixes

  • Fixed weather not refreshing when clicking weather icon on the main page
